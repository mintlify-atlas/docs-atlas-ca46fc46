## Releases

Stellaris Companion follows semantic versioning for all releases. Each release includes new features, bug fixes, and improvements to enhance your strategic gameplay experience.

<Note>
  All releases are published on the [GitHub Releases](https://github.com/gitmaan/stellaris-companion/releases) page, including pre-built binaries for the Rust parser.
</Note>

## Viewing release history

To view detailed release notes and download specific versions:

<Steps>
  <Step title="Visit GitHub Releases">
    Navigate to the [Stellaris Companion Releases](https://github.com/gitmaan/stellaris-companion/releases) page on GitHub.
  </Step>

  <Step title="Browse versions">
    Each release includes:
    - Version number and release date
    - Detailed changelog with features, fixes, and improvements
    - Pre-built binaries for supported platforms
    - Source code archives
  </Step>

  <Step title="Download assets">
    Download the appropriate release assets for your platform, including:
    - Electron application installers
    - Rust parser binaries
    - Source code (zip or tar.gz)
  </Step>
</Steps>

## Release categories

Releases are categorized to help you understand the type of changes:

<AccordionGroup>
  <Accordion title="Features" icon="sparkles">
    New capabilities and functionality added to Stellaris Companion, such as:
    - New analysis modes (Chronicle, Discord integration)
    - Additional game data extraction (DLC support)
    - Enhanced AI advisor capabilities
  </Accordion>

  <Accordion title="Bug fixes" icon="wrench">
    Corrections to existing functionality, including:
    - Crash fixes and stability improvements
    - Platform-specific reliability fixes (Windows, Linux, macOS)
    - Data parsing accuracy improvements
  </Accordion>

  <Accordion title="Improvements" icon="arrow-up">
    Enhancements to existing features, such as:
    - Performance optimizations
    - UI/UX refinements
    - Better error handling and messages
  </Accordion>

  <Accordion title="Breaking changes" icon="triangle-exclamation">
    Changes that may require migration or updates to your setup:
    - API changes
    - Configuration format updates
    - Dependency version updates
  </Accordion>
</AccordionGroup>

## Current development focus

The project is currently prioritizing:

- **DLC coverage**: Ensuring all Stellaris DLC content is properly supported
- **Cross-platform reliability**: Focusing on Windows and Linux stability
- **Polish and stability**: Bug fixes, regression prevention, and improved testing

<Info>
  Net-new feature work is temporarily de-prioritized unless it unblocks a critical bug or platform reliability issue. See the [contribution guide](/contributing/guide) for more details.
</Info>

## Version support

Stellaris Companion follows these support guidelines:

- **Latest stable release**: Fully supported with bug fixes and updates
- **Previous minor versions**: Security fixes only
- **Older versions**: No active support; upgrade recommended

<Warning>
  Always back up your save files before upgrading to a new version, especially during major version updates.
</Warning>
