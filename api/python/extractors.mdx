# Save Extractor

The `stellaris_save_extractor` module provides the `SaveExtractor` class for reading and parsing Stellaris save files.

## SaveExtractor

Main class for extracting data from `.sav` files.

```python
from stellaris_save_extractor import SaveExtractor

with SaveExtractor("/path/to/autosave.sav") as extractor:
    metadata = extractor.get_metadata()
    print(f"Empire: {metadata['name']}")
    print(f"Date: {metadata['date']}")
```

### Military & Fleet Methods

#### get_fleets

Get detailed fleet information.

```python
fleets = extractor.get_fleets()
for fleet in fleets['military_fleets']:
    print(f"{fleet['name']}: {fleet['military_power']} power")
```

#### get_starbases

Get starbase counts and details.

```python
starbases = extractor.get_starbases()
print(f"Citadels: {starbases['citadels']}")
print(f"Total Systems: {starbases['total_systems']}")
```

### Economy Methods

#### get_economy_summary

Get economy overview with production/consumption.

```python
economy = extractor.get_economy_summary()
```

## Extractor Mixins

The `SaveExtractor` is composed of specialized mixin classes:

- **MetadataMixin** - Basic save metadata
- **PlayerMixin** - Player empire status
- **MilitaryMixin** - Fleets and military power
- **EconomyMixin** - Resources and economy
- **DiplomacyMixin** - Diplomatic relations
- **GeographyMixin** - Territory and borders
- **TechnologyMixin** - Research and tech
- **PoliticsMixin** - Factions and government
- **PlanetsMixin** - Planet data
- **SpeciesMixin** - Species information
- **EndgameMixin** - Crisis and endgame content
- **BriefingMixin** - Complete briefing generation

## Example: Analysis Script

```python
from stellaris_save_extractor import SaveExtractor
import json

def analyze_save(save_path: str):
    """Analyze a save file and print key statistics."""
    with SaveExtractor(save_path) as extractor:
        # Get comprehensive briefing
        briefing = extractor.get_complete_briefing()
        
        # Print identity
        identity = briefing['identity']
        print(f"Empire: {identity['empire_name']}")
        print(f"Ethics: {', '.join(identity['ethics'])}")
        print(f"Authority: {identity['authority']}")
        print()
        
        # Print situation
        situation = briefing['situation']
        print(f"Game Date: {briefing['meta']['date']}")
        print(f"Game Phase: {situation['game_phase']}")
        print(f"At War: {situation['at_war']}")
        print()
        
        # Print military
        military = briefing['military']
        print(f"Military Power: {military['total_power']}")
        print(f"Fleet Count: {military['fleet_count']}")
        print()
        
        # Print economy
        economy = briefing['economy']
        print("Monthly Resources:")
        for resource, value in economy['resources_net'].items():
            print(f"  {resource}: {value:+.1f}")
        print()
        
        # Print diplomacy
        diplomacy = briefing['diplomacy']
        print(f"Allies: {len(diplomacy['allies'])}")
        print(f"Rivals: {len(diplomacy['rivals'])}")
        print(f"Wars: {len(diplomacy['wars'])}")

if __name__ == "__main__":
    analyze_save("/path/to/autosave.sav")
```

## Performance Notes

- The extractor uses a Rust-based parser for fast section extraction
- Results are cached per extraction session
- Use context manager (`with` statement) for automatic cleanup
- Call `release_gamestate()` to free memory after extraction
